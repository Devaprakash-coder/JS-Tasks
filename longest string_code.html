Longest String

Definition:

We have to find which string is longest/largest from the given sentence.

Later

Definition:

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Code

      	function findLongestWord(str) {
  		var longestWord = str.split(' ').sort
  		(function(a, b){
  		 return b.length - a.length;
  		 });
  		return longestWord[0];
		}

Result

Assume the given sentance as Apples are red.
The result will show Apple.(Apple is the longest word in the sentance).
